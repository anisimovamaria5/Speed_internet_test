# 🌐 Internet Speed Test

Простой скрипт на Python для замера скорости интернета через последовательные HTTP-запросы к выбранному файлу.

## 📥 Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/anisimovamaria5/Speed_internet_test.git
   
2. Установите зависимости
   ```bash
   pip install -r requirements.txt
   
## 🚀 Использование

  С указанием своего URL
  ```bash
  python speed_test.py https://example.com/...
  ```

📊 Что делает скрипт
 - Делает 10 последовательных GET-запросов к указанному URL
 - Замеряет время каждого запроса
 - Считает объём скачанных данных
 - Вычисляет:
    1. Среднее время запроса
    2. Объем скачанных данных
    3. Среднюю скорость в МБ/с
